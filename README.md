# The KnetMiner Continuous Integration Tools
Utilities to manage builds and Continuous Integration (CI) in KnetMiner (and beyond).

In this repo, we host [the scripts](ci-build-v2/README.md) we use with GitHub Actions to build our projects, usually after each push (that hasn't `[ci skip]`). These are general enough to be imported in any project, including non-KnetMiner ones.

Moreover, the repository has tools and utilities related to project builds, management, testing, CI, and all that jazz.
